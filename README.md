# No eBay Live (Firefox)
![](https://github.com/PinkAvocadoDev/No-eBay-Live-Firefox-Extension/blob/a8417af1e940af081f041d1dc6ff8923dfa3c2ae/assets/no-live-128.png?raw=true)

I'm absolutely tired of eBay Live, it's full of AI slop and scalpers and it takes up a lot of the home page.
This extension hides away all eBay Live sections from the site, except for the actual eBay Live page, as I would assume you would really want to watch those things if you are there, you do you.

So yeah, enjoy your slop-free navigation!
